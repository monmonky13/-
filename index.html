⚙️ GitHub連携設定
✅ GitHub に接続中

GitHub Personal Access Token を登録すると、どのPCからでも同じデータを使えるようになります。

1
こちらをクリック → GitHubのToken発行ページを開く
「Gist」にチェックが入っていることを確認して「Generate token」をクリック
2
表示されたToken（ghp_で始まる文字列）をコピーして下に貼り付ける
3
「保存して連携」ボタンを押す（全PCで同じ手順を繰り返す）
⚠️ Tokenは他人に見せないでください。このデータはこの端末のブラウザにのみ保存され、外部には送信されません（GitHubのAPIのみ使用）。
